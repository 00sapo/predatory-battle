Predatory Battle

Select a BibTex file to upload:

This tool allows you to check if a BibTex file contains references to predatory journals. Predatory journals are journals which claim to be "scientific" whithout providing a proper peer-review process and, often, with a fee-based mechanism through which malevolous publishers make huge profit s.

Citing these papers impoverishes your publication, because no claim from predatory publications can be trusted since no peer-review process is guaranteed. We strongly suggest to always carefully check your journals. See Think. Check. Submit. initiative for a guideline to identify trustworthy journals.

This tool helps you in avoiding any predatory or falsified scientific publication. It checks multiple lists with two different algorithms:

  1. an exact match algorithm, which returns references that with a moderate certainty are predatory
  2. a fuzzy search algorithms that can handle typographic errors, but for which most of the reported publications arenot predatory

The lists used are the following:

  • A list of predatory journals: https://predatoryjournals.com/journals/
  • A list of predatory publishers: https://beallslist.weebly.com/
  • Another list of predatory journals: https://beallslist.weebly.com/standalone-journals.html
  • A list of journals which claims to be indexed in DOAJ, but they are not: https://docs.google.com/spreadsheets/d/1Y_Sza4rPDkf-NNX9kwiErGrKeNTM75md9B63A_gVpaQ/
  • A list of "questionable conferences": https://libguides.caltech.edu/c.php?g=512665&p=3503029
The lists are updated every 30 days.

Unfortunately, we are still not able to extract references from PDF files. However, you can extract a BibTex file using Scholarcy API demo:

  1. Open the Scholarcy page
  2. Select the first "Post" entry
  3. Click on "Try it out"
  4. Upload your PDF file
  5. Click on the "Execute" button at the bottom
Alternatively, you can also copy-and-past your references using this service by Makino Takaki, although you will probably need to correct the copied text.
